# Personal Portfolio 🚀

Welcome to my personal portfolio repository! This project is a reflection of my journey as a Full-Stack Developer, focusing on clean UI, smooth performance, and production-grade code structure.

## 🛠 Tech Stack & Architecture

- **React.js & TypeScript:** I used TypeScript to ensure type safety and write more predictable, bug-free code.
- **Tailwind CSS:** For styling, I utilized Tailwind to build a fully responsive and modern interface without leaving my HTML/JSX.
- **Framer Motion:** All the smooth transitions, scroll animations, and interactive UI elements are powered by Framer Motion.
- **Reusable Components:** The project is built with a modular approach. I’ve developed reusable UI components (Buttons, Cards, Modals) to keep the codebase DRY (Don't Repeat Yourself) and easy to maintain.
